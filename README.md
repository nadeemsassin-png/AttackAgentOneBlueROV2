BlueROV2 Attacker Agent Simulation

C++ simulation of a BlueROV2 UUV retrofitted for a one-way harbor attack mission, built as part of UUV/UAV harbor defense research at UMass Dartmouth.

One of the attacker agents I've researched and presented in a slideshow, using the same specs as it, to run the program.

The agent spawns at an ocean entry point and autonomously moves toward a harbor target across a real Pearl Harbor map loaded from a shapefile.

Part of a larger research project modeling attacker/defender UUV agents in a Pearl Harbor coastal defense scenario, optimized using a genetic algorithm.
